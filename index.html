🙋 Frequently Asked Questions & Quick Guide

⚙️ System & Troubleshooting

❓ Why does the browser flasher say "Failed to download manifest"?

This error means the browser can't locate the dist/manifest.json file. Make sure the dist/ folder containing manifest.json and the binary files exists in the same directory as index.html. Flashing also requires a secure HTTPS context or localhost.

❓ My screen is shifted, scrambled, or showing static after flashing?

This occurs if you flash SSD1306 firmware onto a SH1106 display (or vice versa). SSD1306 is typically for 0.96" screens; SH1106 is for 1.3" screens. Select the correct driver in the builder GUI before compiling, or toggle the #define USE_SH1106 / USE_SSD1306 inside include/Config.h.

❓ Which files are required for my GitHub repository?

To ensure the suite runs correctly from GitHub Pages, you must upload: index.html, omggif.js, build_firmware.py, build_exe.bat, platformio.ini, the src/ and include/ directories, the dist/ directory, and OLO_Show_Builder.exe.

❓ My COM port isn't showing up. What should I do?

Ensure you are using a USB cable that supports data transfer (many are charge-only). Check that the CH340 or CP210x drivers are installed on your computer. If using the ESP32-C3 Supermini, hold down the BOOT button while plugging in the USB cable to enter bootloader mode.

🎬 Animation & Operation

🚀 Quick Guide: How to Use the Converter
  1. Navigate to the Animation Converter tab above.
  2. Click the folder icon next to a slot to select a video/GIF, or click 📦 Load ZIP of Videos to import multiple files at once.
  3. Use the ➕ Add Slot button if you have more animations to load (up to 10 slots).
  4. Adjust conversion settings (Dithering algorithm, scale mode, target FPS) to preview the output frame.
  5. Click 🚀 Convert & Process All Loaded Videos.
  6. Click 📦 Download All as ZIP. Extract and compile these headers using the firmware builder!
❓ What are the touch controls and gestures?

Single tap to Play/Pause. Double tap to cycle to the next slot (or the clock). Long press (1 second) to restart the current animation back to the first frame.

❓ Can I change the default pin wiring?

Yes, the pins can be customized in include/Config.h before compiling your custom firmware. By default, OLED SDA is GPIO 20, SCL is GPIO 21, Touch is GPIO 1, Buzzer is GPIO 2, and NeoPixel is GPIO 6.
